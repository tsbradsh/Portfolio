Refs: `const canvasRef = useRef<HTMLCanvasElement>(null)` — passed to JSX canvas element, then to `new THREE.WebGLRenderer({ canvas: canvasRef.current, antialias: true })` (per Pattern 3 — JSX ref, not document.body.appendChild).

Single `useEffect(() => { ... }, [])` containing ALL of the following (per D-05):

1. **Guard:** `if (!canvasRef.current) return` (per Pitfall 4)

2. **Renderer init with try/catch (per D-13, WEBGL-04):**
   ```typescript
   let renderer: THREE.WebGLRenderer
   try {
     renderer = new THREE.WebGLRenderer({ canvas: canvasRef.current, antialias: true })
   } catch {
     setWebglAvailable(false)
     return
   }
   ```

3. **Scene setup** — port verbatim from `_archive/scriptGL.js` lines 2-4:
   ```typescript
   const scene = new THREE.Scene()
   const camera = new THREE.OrthographicCamera(-1, 1, 1, -1, 0.1, 10)

   ```

4. **Renderer config (per WEBGL-05, D-16 discretion):**
   ```typescript
   renderer.setSize(window.innerWidth, window.innerHeight)
   renderer.setPixelRatio(Math.min(window.devicePixelRatio, 1.5))
   ```

5. **Geometry + ShaderMaterial** — port from `_archive/scriptGL.js` lines 10-59. The fragment shader string MUST be copied verbatim (all GLSL including `precision highp float`, `mandelbrot()`, `hsv2rgb()`, `main()`). Uniforms:
   ```typescript
   const uniforms = {
     uZoom: { value: 1.0 },
     uPan: { value: new THREE.Vector2(-0.743, 0.11) },
     uHueOffset: { value: 0.0 },
     uResolution: { value: new THREE.Vector2(window.innerWidth, window.innerHeight) },
   }
   const geometry = new THREE.PlaneGeometry(2, 2)
   const material = new THREE.ShaderMaterial({ uniforms, fragmentShader: FRAGMENT_SHADER })

   ```
   Define `FRAGMENT_SHADER` as a `const` string above the component (module-level) containing the exact GLSL from `_archive/scriptGL.js` lines 18-58.

6. **Mutable animation refs** — plain objects inside effect closure (per D-04, not module-level per Pitfall 2):
   ```typescript
   let zoom = 1.0
   let targetZoom = 1.0
   let hueOffset = 0.0
   let targetHueOffset = 0.0
   let animFrameId: number
   ```

7. **prefers-reduced-motion** — read INSIDE useEffect only (per Pitfall 3):
   ```typescript
   const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches
   ```

8. **Scroll listener (per D-04, D-05, D-07, D-08):**
   ```typescript
   const onScroll = () => {
     const scrollY = window.scrollY
     targetZoom = Math.exp(scrollY / 1300)
     targetHueOffset = (scrollY / 15000) % 1
   }
   window.addEventListener('scroll', onScroll, { passive: true })
   ```

9. **Resize handler:**
   ```typescript
   const onResize = () => {
     renderer.setSize(window.innerWidth, window.innerHeight)
     uniforms.uResolution.value.set(window.innerWidth, window.innerHeight)
   }
   window.addEventListener('resize', onResize)
   ```

10. **Animation loop (per D-08):**
    ```typescript
    function animate() {
      animFrameId = requestAnimationFrame(animate)
      zoom += (targetZoom - zoom) * 0.05
      hueOffset += (targetHueOffset - hueOffset) * 0.01
      uniforms.uZoom.value = zoom
      uniforms.uHueOffset.value = hueOffset
      renderer.render(scene, camera)
    }
    ```

11. **Start animation or static render (per D-16):**
    ```typescript
    if (!prefersReducedMotion) {
      animate()
    } else {
      renderer.render(scene, camera)
    }
    ```

12. **Visibility change handler:**
    ```typescript
    const onVisibilityChange = () => {
      if (document.hidden) {
        cancelAnimationFrame(animFrameId)
      } else if (!prefersReducedMotion) {
        animate()
      }
    }
    document.addEventListener('visibilitychange', onVisibilityChange)
    ```

13. **Intersection Observer (per WEBGL-05) — threshold 0:**
    ```typescript
    const observer = new IntersectionObserver(
      (entries) => {
        const [entry] = entries
        if (!entry.isIntersecting) {
          cancelAnimationFrame(animFrameId)
        } else if (!prefersReducedMotion && !document.hidden) {
          animate()
        }
      },
      { threshold: 0 }
    )
    observer.observe(canvasRef.current)
    ```

14. **Cleanup return (per D-09, D-10) — MUST dispose all resources:**
    ```typescript
    return () => {
      cancelAnimationFrame(animFrameId)
      window.removeEventListener('scroll', onScroll)
      window.removeEventListener('resize', onResize)
      document.removeEventListener('visibilitychange', onVisibilityChange)
      observer.disconnect()
      scene.remove(mesh)
      geometry.dispose()
      material.dispose()
      renderer.dispose()
    }
    ```
